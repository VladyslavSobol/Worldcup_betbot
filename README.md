# Бот Фан-Ставок На ЧС-2026

Приватний Telegram-бот для фан-ставок між друзями під час FIFA World Cup 2026.

## Можливості

- Нові користувачі отримують тестовий баланс `$100.00`.
- Ринки: 1X2, тотали, фори, довгострокові та ручні адмін-ринки.
- Сума ставки списується одразу, коефіцієнт фіксується на момент ставки.
- Мінімальна ставка `$1.00`; максимального ліміту ставки немає, окрім доступного балансу.
- Таблиця лідерів рахує банкрол: доступний баланс плюс відкриті ставки.
- Адмін-команди для розрахунку, скасування й закриття ринків.
- Інтеграція з The Odds API для коефіцієнтів, із ручним fallback.
- PostgreSQL для запуску, SQLite для тестів.

## Швидкий Старт

1. Скопіюй `.env.example` у `.env`.
2. Заповни `TELEGRAM_BOT_TOKEN`, `ADMIN_TELEGRAM_IDS` і `ODDS_API_KEY`.
3. Запусти через Docker:

```bash
docker compose up --build
```

Або локально:

```bash
python -m venv .venv
. .venv/Scripts/activate
pip install -e ".[dev]"
python -m app.main
```

## Основні Команди

- `/start` - створити акаунт і отримати тестові `$100.00`.
- `/help` - показати підказку по командах.
- `/matches` - найближчі матчі ЧС.
- `/markets <id_матчу>` - ринки для матчу.
- `/bet <id_коеф> <сума>` - зробити фан-ставку.
- `/mybets` - твої ставки.
- `/leaderboard` - таблиця лідерів.
- `/admin_settle <id_матчу> <голи_1> <голи_2>` - розрахувати матч.
- `/admin_void <id_матчу> [причина]` - скасувати відкриті ставки.
- `/admin_close <id_матчу>` - закрити ринки матчу.
- `/admin_reset` - видалити всі ставки й повернути всім стартовий баланс.
- `/admin_sync` - одразу оновити лінії.
- `/bind_group` - прив'язати груповий чат для анонсів ставок і результатів.

## Груповий Чат

1. Додай бота в Telegram-групу.
2. Адмін пише в групі `/bind_group`.
3. Бот запам'ятовує цей чат як головний.
4. У групі працюють кнопки `Матчі`, `Лідерборд`, `Як граємо?`.
5. Коли хтось робить ставку в приваті або іншому чаті, бот публікує анонс у прив'язану групу.
6. Після `/admin_settle` або `/admin_void` бот публікує результат і оновлений лідерборд.

За замовчуванням live-ринки з провайдера: `h2h,spreads,totals`.
Довгострокові ринки можна додавати вручну або вмикати окремо, коли провайдер стабільно віддає їх для турніру.
Ринок `Обидві заб'ють` вимкнений, бо The Odds API не віддає для нього реальні коефіцієнти на поточному World Cup sport key.

## Примітки

Бот тільки для приватної гри на тестовий баланс. Він не приймає депозити,
виведення коштів або реальні грошові ставки.
